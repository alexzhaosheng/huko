huko-explicit

Minimalist CLI. Maximalist intelligence. Configurable all the way down.

[01] Full Power, Lean Footprint

Full mode — a production-grade agent with full planning capabilities, complete tool suite, and project context. Built for complex, multi-step workflows.

Lean mode — bash-only, ~400 token overhead, fast even on small models. Perfect for one-shot questions and simple operations.

# full — complete agent: planning, 13 tools, project context, role system
huko -- "review this diff, flag risky changes, suggest fixes"
# lean — bash-only, ~400 token overhead, fast even on small models
huko --lean -- "what's the uptime of this machine"

[02] Project as Context

State lives in the project. Sessions, config, and agent memory are scoped to the directory you're in — .huko/, just like .git/. CD into a repo and huko picks up its full context. CD out and it's a different world.

No trace when you want none. Pass --memory to skip session storage entirely — nothing written to disk, perfect for one-off questions and private queries.

# state lives in .huko/ — scoped to this project, just like git
cd ~/projects/backend && huko -- "continue the auth refactor"
# switch directories, switch worlds — different sessions, different config
cd ~/projects/frontend && huko -- "review the new component"
# one-off, no trace — --memory skips session storage entirely
huko --memory -- "explain this regex"

[03] Layered Config

# inspect the resolved runtime config
huko config show
# inspect the merged infra view
huko info
# inspect only the global layer
huko info global

[04] Project Memory

# continue work inside this repo
huko -- continue the refactor plan
# list saved sessions for this project
huko sessions list
# branch into a fresh thread
huko --new --title="release triage" -- summarize today's blockers

[05] Zero-trace Runs

# run without touching session storage
huko --memory -- "summarize this build log in 3 bullets"
# return machine-readable output
huko --memory --json -- "return a release note draft"
# ask something private and leave no trail
huko --memory -- "private question"

[06] Inspectable Agents

# capture a final result as JSON
huko --json -- "generate a changelog from these commits"
# stream semantic events line by line
huko --jsonl -- "fetch the page and extract the title"
# inspect the project-specific agent surface
huko info project
